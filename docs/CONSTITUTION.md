# Constitution

## What This Is

`dbl-policy-gates` provides the deterministic gate algebra used to compose
governance functions for `dbl-policy`.

## Governing Principles

1. Gates are pure and deterministic.
2. Gates consume authoritative inputs only.
3. Gates do not carry policy identity.
4. Only root policies emit `PolicyDecision`.
5. Describe output must be stable and digestible.

## What This Is Not

- Not a rule engine.
- Not a policy registry.
- Not an execution surface.
- Not an observational analytics layer.
